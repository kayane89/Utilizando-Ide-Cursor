# Utilizando-Ide-Cursor

Este projeto é uma aplicação web simples desenvolvida em HTML5, CSS3 e JavaScript puro, focada no aprendizado de manipulação de DOM e validação de entrada de dados.

## 🚀 Funcionalidades

- **Gerador de Tabuada:** Calcula automaticamente a tabuada de 1 a 20 do número inserido.
- **Validação Robusta:** - Impede a entrada de números negativos.
  - Impede caracteres especiais (pontos, vírgulas).
  - Bloqueia a entrada de letras (caractere 'e' científico).
- **Interatividade:** Suporta acionamento via botão "Gerar" ou tecla "Enter".

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Layout moderno com Grid e design responsivo.
- **JavaScript (Vanilla):** Lógica de negócio e validação de eventos.

## 📂 Como executar

1. Salve os arquivos `index.html`, `estilo.css` e `script.js` na mesma pasta.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
